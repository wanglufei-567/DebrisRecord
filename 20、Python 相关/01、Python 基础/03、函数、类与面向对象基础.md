## Python 函数、类与面向对象基础

### 一、函数定义与返回值

#### 1.1、语法与用法

函数使用 `def` 定义，函数体通过缩进表达

- `def name(args):`：定义函数
- `return value`：返回结果
- 不写 `return` 时默认返回 `None`
- `-> str` 这类语法表示返回值类型

#### 1.2、示例代码

```python
# Python
def normalize_tool_name(name: str) -> str:
    return name.strip().lower()

def log_tool_call(name: str) -> None:
    print(f"call tool: {name}")
```

```js
// JavaScript
function normalizeToolName(name) {
  return name.trim().toLowerCase();
}

function logToolCall(name) {
  console.log(`call tool: ${name}`);
}
```

### 二、默认参数与可变默认值

#### 2.1、语法与用法

默认参数写在函数参数列表里，表示调用者不传这个参数时使用的默认值

```python
def run_task(timeout: int = 30) -> None:
    ...
```

需要特别注意的是，不要把可变对象直接写成默认值，例如 `tools: list[str] = []`

因为 **Python** 的默认参数是在函数定义时创建的，不是每次调用函数时重新创建

例如👇：

```python
def collect_tools(tool: str, tools: list[str] = []) -> list[str]:
    tools.append(tool)
    return tools
```

第一次调用后，默认列表变成 `["read_file"]`

```python
collect_tools("read_file")
```

第二次调用时，如果仍然不传 `tools`，**Python** 不会创建新列表，而是继续复用上一次的默认列表

```python
collect_tools("write_file")
# ["read_file", "write_file"]
```

这通常不是预期结果，因为两次调用之间的数据被意外共享了

更稳妥的写法是用 `None` 表示“调用者没有传列表”，然后在函数内部创建一个新的空列表

#### 2.2、示例代码

```python
# Python
def collect_tools(tool: str, tools: list[str] | None = None) -> list[str]:
    if tools is None:
        tools = []

    tools.append(tool)
    return tools
```

```js
// JavaScript
function collectTools(tool, tools = []) {
  tools.push(tool);
  return tools;
}
```

### 三、关键字参数、`*args` 与 `**kwargs`

#### 3.1、语法与用法

**Python** 调用函数时，既可以按位置传参，也可以按参数名传参

按位置传参时，参数靠顺序匹配

```python
run_task("gpt-example", 30)
```

按关键字传参时，参数靠名字匹配

```python
run_task(model="gpt-example", timeout=30)
```

这两种方式可以混用，但通常要遵守一个顺序：位置参数在前，关键字参数在后

```python
run_task("gpt-example", timeout=30)
```

`*args` 和 `**kwargs` 是在函数定义阶段接收“多余参数”的两种写法

- `*args`：收集多余位置参数，结果是 `tuple`
- `**kwargs`：收集多余关键字参数，结果是 `dict`

#### 3.2、参数说明

以 `def call_provider(model: str, *messages: dict, **options: object)` 为例

- `model`：普通位置参数
- `*messages`：收集 `model` 后面多出来的位置参数
- `**options`：收集多出来的关键字参数

调用时可以这样写

```python
call_provider(
    "gpt-example",
    {"role": "user", "content": "hello"},
    {"role": "assistant", "content": "hi"},
    timeout=30,
    temperature=0.7,
)
```

参数绑定结果是

- `model == "gpt-example"`
- `messages == ({"role": "user", "content": "hello"}, {"role": "assistant", "content": "hi"})`
- `options == {"timeout": 30, "temperature": 0.7}`

也就是说，**Python** 会先匹配普通参数，再把多余的位置参数交给 `*messages`，最后把多余的关键字参数交给 `**options`

#### 3.3、示例代码

```python
# Python
def call_provider(model: str, *messages: dict, **options: object) -> dict:
    return {
        "model": model,
        "messages": list(messages),
        "options": options,
    }


result = call_provider(
    "gpt-example",
    {"role": "user", "content": "hello"},
    {"role": "assistant", "content": "hi"},
    timeout=30,
    temperature=0.7,
)
```

```js
// JavaScript
function callProvider(model, messages, options = {}) {
  return {
    model,
    messages,
    options,
  };
}

const result = callProvider(
  "gpt-example",
  [{ role: "user", content: "hello" }],
  { timeout: 30 },
);
```

### 四、class、self 与实例属性

#### 4.1、语法与用法

- `class Name:`：定义类
- `__init__`：构造函数
- `self`：当前实例，类似 **JavaScript** 的 `this`，但必须显式写在方法参数里
- `self.name = name`：写入实例属性
- 类属性定义在类体中，实例属性通常定义在 `__init__` 中

#### 4.2、示例代码

```python
# Python
class ModelProvider:
    provider_type = "llm"

    def __init__(self, model: str):
        self.model = model

    def complete(self, prompt: str) -> str:
        return f"{self.model}: {prompt}"
```

```js
// JavaScript
class ModelProvider {
  static providerType = "llm";

  constructor(model) {
    this.model = model;
  }

  complete(prompt) {
    return `${this.model}: ${prompt}`;
  }
}
```

### 五、继承、重写与 super

#### 5.1、语法与用法

- `class Child(Base):`：继承父类
- 子类定义同名方法就是重写
- `super()` 用于调用父类实现

#### 5.2、示例代码

```python
# Python
class BaseTool:
    def run(self, argument: str) -> str:
        return argument

class UppercaseTool(BaseTool):
    def run(self, argument: str) -> str:
        raw = super().run(argument)
        return raw.upper()
```

```js
// JavaScript
class BaseTool {
  run(argument) {
    return argument;
  }
}

class UppercaseTool extends BaseTool {
  run(argument) {
    const raw = super.run(argument);
    return raw.toUpperCase();
  }
}
```

### 六、property

#### 6.1、语法与用法

`@property` 把一个无参数方法暴露成属性读取形式

- 适合计算派生值
- 适合隐藏内部字段命名
- setter 和 cached property 后续补充

#### 6.2、示例代码

```python
# Python
class Session:
    def __init__(self, session_id: str):
        self.session_id = session_id

    @property
    def storage_key(self) -> str:
        return f"session:{self.session_id}"
```

```js
// JavaScript
class Session {
  constructor(sessionId) {
    this.sessionId = sessionId;
  }

  get storageKey() {
    return `session:${this.sessionId}`;
  }
}
```
