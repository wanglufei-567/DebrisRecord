## 大文件上传

大文件上传最成熟的解决方案是：**文件切片（Chunking） + 并发控制 + 断点续传（Breakpoint Resumption）**。

核心原理：将大文件拆分为多个小块（**Chunks**），利用 `Blob.prototype.slice()` 方法进行切割

1. **文件切片**：通过 `Blob.slice` 将大文件拆分成固定大小（例如 2MB）的 `Blob` 对象
2. **生成 Hash**：使用 `SparkMD5` 计算文件的唯一标识，用于服务端区分文件及实现秒传
3. **分片上传**：通过 `FormData` 将每个分片上传至服务器
4. **合并请求**：所有分片上传完成后，通知服务端合并分片

完整代码实现👇：

```JavaScript
const CHUNK_SIZE = 1024*1024*3 // 3MB
const UPLOAD_URL = ''
const MERGE_URL = ''

// 创建文件切片
const createChunks = (file) => {
  const chunks = []
  let cur = 0
  while(cur < file.size) {
    const chunk = file.slice(cur, cur + CHUNK_SIZE)
    chunks.push(chunk)
    cur += CHUNK_SIZE
  }
  return chunks
}

// 上传文件切片
const uploadChunks = async(chunks, fileHash, limit = 3) => {
  // 任务队列
  const tasks = new Set()

  for(let i = 0; i < chunks.length; i++) {
    // 构建切片上传信息，包含切片内容和文件hash
    const createRequest = () => {
      const formData = new FormData();
      formData.append('index', i);
      formData.append('file', chunks[i]);
      formData.append('fileHash', fileHash);
      return fetch(UPLOAD_URL, { method: 'POST', body: formData });
    };

    // 构建上传任务
    const task = createRequest().then(res => {
      // 第一次尝试：如果成功，直接返回结果
      if (res.ok) return res;

      // 如果第一次失败，进行重试，返回一个新的 Promise，重新构建上传任务
      return createRequest();
    }).then(res => {
      // 处理最终结果无论是第一次的结果，还是重试后的结果
      if (!res.ok) {
        throw new Error(`切片 ${i} 上传失败`);
      }

      return res;
    }).finally(() => tasks.delete(task));

    // 将任务加入任务队列
    tasks.add(task)

    // 如果任务队列中的任务数大于等于最大并发数，则等待其中一个任务完成
    if(tasks.size >= limit) {
      await Promise.race(tasks)
    }
  }

  // 等待所有任务完成
  await Promise.all(tasks)

  // 合并切片
  await fetch(MERGE_URL, {
    method: 'POST',
    body: JSON.stringify({
      fileHash: fileHash
    })
  })
}

// 监听文件选择
const fileInput = document.querySelector('input[type="file"]')
fileInput.onchange = (e) => {
  // 获取文件
  const file = e.target.files[0]
  // 创建文件切片
  const chunks = createChunks(file)
  // 上传文件切片
  uploadChunks(chunks, 'fileHash')
}
```
